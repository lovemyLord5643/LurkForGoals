# MERN app

This is the app from the [Learn the MERN Stack](https://www.youtube.com/watch?v=-0exw-9YJBo) series on YouTube.

## Usage

Use your own ```.env``` in the backend folder

### Install dependencies

```
# Backend deps
cd backend
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server

```
# Backend server
cd backend
npm run server

# Frontnd server
cd Frontend
npm start
```
